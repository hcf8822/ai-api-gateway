接入文档

快速开始

1. 在「API 令牌」页面创建一个令牌，获取 sk-xxx 格式的 Key

2. 使用以下配置调用 API：

from openai import OpenAI

client = OpenAI(
    api_key="sk-你的令牌",
    base_url="https://ai-api-gateway-dmyn.onrender.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)

cURL 示例

curl https://ai-api-gateway-dmyn.onrender.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-你的令牌" \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
